#### Array,Dictionary
```
var fruits = ["strawberries", "limes", "tangerines"]
fruits[1] = "grapes"

var occupations = [
    "Malcolm": "Captain",
    "Kaylee": "Mechanic",
 ]
occupations["Jayne"] = "Public Relations"
```
#### want to add elements automatically
```
fruits.append("blueberries")
```
#### use for brackets
```
// Array
fruits = []
// Dictionary
occupations = [:]

```
